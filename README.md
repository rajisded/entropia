# ENTROPIA

A sovereign intellectual civilization.

## Overview
This project is a high-end Next.js application representing the digital interface of Entropia.

## Architecture
- **Framework**: Next.js 15 (App Router)
- **Styling**: CSS Modules / Global CSS variables (No Tailwind required, high performance)
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Type Safety**: TypeScript

## Project Structure
- `src/app/page.tsx`: Main landing page composed of section components.
- `src/app/prospectus/page.tsx`: Specialized confidential document view.
- `src/components/`: Reusable UI components (Hero, Vision, Pillars, etc.).
- `src/app/globals.css`: Core design system variables and utilities.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Aesthetic Direction
Designed for "elite cognition" and "technological acceleration" with a dark, high-contrast, premium interface.
